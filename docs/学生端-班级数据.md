# 学生端 - 班级数据

## 功能概述

学生端班级数据模块提供学习数据统计、可视化图表和班级排名功能，帮助学生了解自己的学习进度和班级排名情况。

## 页面功能

### 1. 顶部数据展示
- 累计查词量
- 任务完成数
- 总错题数

### 2. 中间可视化图表
- 每日学习数据折线图（近7天）
- 班级 vs 个人完成率走势（近8周）

### 3. 底部班级排名
- 按学习时长排名
- 按掌握单词量排名
- 支持切换排名方式

## 接口列表

### 1. 获取顶部统计数据

**接口地址**：`GET /api/student-class-data/statistics`

**功能说明**：获取学生班级数据的顶部统计信息。

**请求参数**：无

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "totalWords": 256,
    "completedTasks": 8,
    "totalErrors": 15
  }
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| totalWords | number | 累计查词量 |
| completedTasks | number | 任务完成数 |
| totalErrors | number | 总错题数 |

---

### 2. 获取每日学习数据

**接口地址**：`GET /api/student-class-data/daily-study`

**功能说明**：获取近7天的每日学习数据，用于绘制折线图。

**请求参数**：无

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "dates": ["04-15", "04-16", "04-17", "04-18", "04-19", "04-20", "04-21"],
    "wordCounts": [12, 18, 15, 22, 16, 20, 14],
    "taskCounts": [2, 3, 2, 4, 3, 2, 3],
    "errorCounts": [3, 2, 4, 2, 3, 1, 2],
    "studyMinutes": [45, 60, 50, 75, 55, 65, 48]
  }
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| dates | string[] | 日期标签（近7天） |
| wordCounts | number[] | 每日查词量 |
| taskCounts | number[] | 每日完成任务数 |
| errorCounts | number[] | 每日错题数 |
| studyMinutes | number[] | 每日学习时长（分钟） |

---

### 3. 获取班级 vs 个人完成率走势

**接口地址**：`GET /api/student-class-data/compare`

**功能说明**：获取近8周的班级平均完成率与个人完成率对比数据，用于绘制对比折线图。

**请求参数**：无

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "weeks": ["第1周", "第2周", "第3周", "第4周", "第5周", "第6周", "第7周", "第8周"],
    "classRate": [45, 52, 58, 63, 70, 75, 82, 88],
    "myRate": [50, 58, 65, 72, 80, 88, 95, 102]
  }
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| weeks | string[] | 周次标签（近8周） |
| classRate | number[] | 班级平均完成率（%） |
| myRate | number[] | 个人完成率（%） |

---

### 4. 获取班级排名

**接口地址**：`GET /api/student-class-data/ranking`

**功能说明**：获取班级学生排名列表，支持按学习时长或掌握单词量排序。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| type | string | 否 | 排名类型：time（按学习时长，默认）/ words（按掌握单词量） |

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "list": [
      {
        "rank": 1,
        "name": "王小明",
        "userId": 101,
        "studyTime": 120,
        "masteredWords": 520,
        "isMe": false
      },
      {
        "rank": 5,
        "name": "张三",
        "userId": 1,
        "studyTime": 85,
        "masteredWords": 400,
        "isMe": true
      }
    ],
    "maxStudyTime": 120,
    "maxWords": 520,
    "total": 15
  }
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| list | array | 排名列表 |
| list[].rank | number | 排名 |
| list[].name | string | 学生姓名 |
| list[].userId | number | 用户ID |
| list[].studyTime | number | 学习时长（小时） |
| list[].masteredWords | number | 掌握单词量 |
| list[].isMe | boolean | 是否是当前用户 |
| maxStudyTime | number | 最大学习时长（用于进度条计算） |
| maxWords | number | 最大单词量（用于进度条计算） |
| total | number | 总人数 |
