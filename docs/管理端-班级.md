# 管理端 - 班级接口文档

## 概述

管理端班级相关接口，提供班级审核、班级管理功能。

**基础路径**: `/api/admin-home`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/class-review/list` | GET | 获取待审核班级列表（支持搜索） |
| 2 | `/class-review/management-list` | GET | 获取已通过班级列表（支持搜索） |
| 3 | `/class-review/approve/:classId` | POST | 审核通过班级 |
| 4 | `/class-review/reject/:classId` | POST | 拒绝班级 |
| 5 | `/class-review/delete/:classId` | DELETE | 删除班级 |

---

## 接口详情

### 1. 获取待审核班级列表（支持搜索）

**接口地址**: `GET /api/admin-home/class-review/list`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| teacherName | String | 否 | 老师名字（模糊搜索） |
| pageNum | Number | 否 | 页码，默认 1 |
| pageSize | Number | 否 | 每页条数，默认 10 |

**响应示例**:
```json
{
  "code": 200,
  "rows": [
    {
      "classId": 101,
      "className": "A级-英语强化班",
      "classLevel": "A",
      "maxStudents": 30,
      "taskCount": 15,
      "createTime": "2026-04-18",
      "teacherId": "teacher",
      "teacherName": "张老师"
    }
  ],
  "total": 3
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| classId | Number | 班级ID |
| className | String | 班级名称 |
| classLevel | String | 班级等级 (A/B/C/D) |
| maxStudents | Number | 最大学生数 |
| taskCount | Number | 任务数量 |
| createTime | String | 创建时间 |
| teacherId | String | 老师ID |
| teacherName | String | 老师名字 |

---

### 2. 获取已通过班级列表（支持搜索）

**接口地址**: `GET /api/admin-home/class-review/management-list`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| teacherName | String | 否 | 老师名字（模糊搜索） |
| classLevel | String | 否 | 班级等级 (A/B/C/D) |
| pageNum | Number | 否 | 页码，默认 1 |
| pageSize | Number | 否 | 每页条数，默认 10 |

**响应示例**:
```json
{

  "rows": [
    {
      "classId": 1,
      "className": "A级-英语精英班",
      "classLevel": "A",
      "currentStudents": 25,
      "maxStudents": 30,
      "taskCount": 15,
      "createTime": "2026-03-01",
      "teacherId": "teacher",
      "teacherName": "张老师"
    }
  ],
  "total": 4
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| classId | Number | 班级ID |
| className | String | 班级名称 |
| classLevel | String | 班级等级 (A/B/C/D) |
| currentStudents | Number | 当前学生数 |
| maxStudents | Number | 最大学生数 |
| taskCount | Number | 任务数量 |
| createTime | String | 创建时间 |
| teacherId | String | 老师ID |
| teacherName | String | 老师名字 |

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
