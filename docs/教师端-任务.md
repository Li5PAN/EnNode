# 教师端 - 任务接口文档

## 概述

教师端任务管理相关接口，提供任务列表查询、详情查看、督促提醒等功能。

**基础路径**: `/api/teacher-home`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/task/list` | GET | 获取任务列表（支持等级筛选） |
| 2 | `/task/:taskId` | GET | 获取任务详情（包含题目列表） |
| 3 | `/task/:taskId/remind` | POST | 督促学生完成任务 |

---

## 接口详情

### 1. 获取任务列表（支持等级筛选）

**接口地址**: `GET /api/teacher-home/task/list`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| classLevel | String | 否 | 班级等级 (A/B/C/D) |
| pageNum | Number | 否 | 页码，默认 1 |
| pageSize | Number | 否 | 每页条数，默认 10 |

**响应示例**:
```json
{
  "code": 200,
  "rows": [
    {
      "id": 1,
      "taskName": "第三单元单词测试",
      "classLevel": "A",
      "className": "A级-英语精英班",
      "questionCount": 50,
      "startTime": "2026-04-15 08:00:00",
      "deadline": "2026-04-25 23:59:59",
      "completedCount": 20,
      "totalStudents": 25,
      "createTime": "2026-04-14 10:00:00"
    }
  ],
  "total": 6
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| id | Number | 任务ID |
| taskName | String | 任务名称 |
| classLevel | String | 班级等级 (A/B/C/D) |
| className | String | 班级名称 |
| questionCount | Number | 题目数量 |
| startTime | String | 开始时间 |
| deadline | String | 截止时间 |
| completedCount | Number | 已完成人数 |
| totalStudents | Number | 总学生数 |
| createTime | String | 发布时间 |

---

### 2. 获取任务详情（包含题目列表）

**接口地址**: `GET /api/teacher-home/task/:taskId`

**请求头**:
```
Authorization: Bearer <token>
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|:----:|------|
| taskId | Number | 任务ID |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "id": 1,
    "taskName": "第三单元单词测试",
    "classLevel": "A",
    "className": "A级-英语精英班",
    "questionCount": 50,
    "startTime": "2026-04-15 08:00:00",
    "deadline": "2026-04-25 23:59:59",
    "completedCount": 20,
    "totalStudents": 25,
    "createTime": "2026-04-14 10:00:00",
    "questions": [
      {
        "type": "choice",
        "content": "What is the meaning of \"beautiful\"?",
        "options": ["美丽的", "丑陋的", "普通的", "奇怪的"],
        "correctIndexes": [0]
      },
      {
        "type": "fill-blank",
        "content": "The opposite of \"fast\" is ___.",
        "answer": "slow"
      },
      {
        "type": "spell",
        "content": "请写出\"学习\"对应的英文单词",
        "answer": "study"
      }
    ]
  }
}
```

**题目字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| type | String | 题目类型 (choice/fill-blank/spell) |
| content | String | 题目内容 |
| options | Array | 选项列表（选择题） |
| correctIndexes | Array | 正确答案索引（选择题） |
| answer | String | 正确答案（填空题/单词拼写） |

---

### 3. 督促学生完成任务

**接口地址**: `POST /api/teacher-home/task/:taskId/remind`

**请求头**:
```
Authorization: Bearer <token>
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|:----:|------|
| taskId | Number | 任务ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "督促提醒已发送至 5 位未完成作业的学生"
}
```

---

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
