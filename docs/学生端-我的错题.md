# 学生端 - 我的错题

## 功能概述

学生端错题模块提供错题管理功能，包括错题列表查看、筛选、导入导出、批量删除等操作。帮助学生系统性地管理和复习错题。

## 页面功能

### 1. 错题列表
- 展示所有错题，支持分页
- 显示题目内容、类型、正确答案、用户答案、所属任务、错误日期、掌握状态
- 支持按题目类型和日期范围筛选
- 支持批量选择和删除

### 2. 错题详情
- 查看错题详细信息
- 显示题目内容、正确答案、用户答案、解析

### 3. 导入导出
- 支持Excel格式导入错题
- 支持导出为Excel或PDF格式
- 提供导入模板下载

**接口地址**：`GET /api/student-errors/overview`

**功能说明**：获取当前学生的错题统计数据。

    "typeDistribution": [

**字段说明**：

### 2. 获取错题列表

**接口地址**：`GET /api/student-errors/list`

**功能说明**：获取当前学生的错题列表，支持筛选和分页。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| questionType | string | 否 | 题目类型：1-选择题，2-单词拼写，3-填空题 |
| startDate | string | 否 | 开始日期（YYYY-MM-DD） |
| endDate | string | 否 | 结束日期（YYYY-MM-DD） |
| pageNum | number | 否 | 页码，默认1 |
| pageSize | number | 否 | 每页条数，默认10 |

**响应示例**：
```json
{
  "code": 200,
  "rows": [
    {
      "wrongId": 1,
      "questionContent": "The teacher asked us to _______ the new words after class.",
      "questionType": "1",
      "questionTypeName": "选择题",
      "correctAnswer": "A. memorize",
      "userAnswer": "B. remember",
      "taskName": "第三单元单词测试",
      "taskId": 101,
      "wrongDate": "2026-04-18",
      "isMastered": "0",
      "explanation": "memorize强调有意识地记忆，remember强调回忆起某事。"
    }
  ],
  "total": 12
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| wrongId | number | 错题ID |
| questionContent | string | 题目内容 |
| questionType | string | 题目类型：1-选择题，2-单词拼写，3-填空题 |
| questionTypeName | string | 题目类型名称 |
| correctAnswer | string | 正确答案 |
| userAnswer | string | 用户答案 |
| taskName | string | 所属任务名称 |
| taskId | number | 所属任务ID |
| wrongDate | string | 错误日期 |
| isMastered | string | 是否已掌握：0-未掌握，1-已掌握 |
| explanation | string | 题目解析 |

### 9. 导出错题

**接口地址**：`GET /api/student-errors/export`

**功能说明**：导出错题为Excel或PDF格式，支持导出全部或筛选后的错题。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| format | string | 是 | 导出格式：excel / pdf |
| wrongIds | string | 否 | 指定导出的错题ID，多个用逗号分隔 |
| questionType | string | 否 | 题目类型筛选：1-选择题，2-单词拼写，3-填空题 |
| startDate | string | 否 | 开始日期（YYYY-MM-DD） |
| endDate | string | 否 | 结束日期（YYYY-MM-DD） |

**响应说明**：
- 成功时直接返回文件流
- Content-Type: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` (Excel)
- Content-Type: `application/pdf` (PDF)
- Content-Disposition: `attachment; filename="student_errors_export.xlsx"` 或 `.pdf`

**响应示例（失败）**：
```json
{
  "code": 400,
  "msg": "请指定导出格式（excel/pdf）"
}
```

```json
{
  "code": 400,
  "msg": "暂无错题可导出"
}
```
