# 管理员端 - 人员管理接口文档

## 概述

管理员端人员管理相关接口，提供人员列表查询、详情查看、删除等功能。

**基础路径**: `/api/admin-home`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/user/list` | GET | 获取人员列表（支持身份筛选、搜索） |
| 2 | `/user/:userId` | GET | 获取人员详情 |
| 3 | `/user/:userId` | DELETE | 删除人员 |

---

## 接口详情

### 1. 获取人员列表（支持身份筛选、搜索）

**接口地址**: `GET /api/admin-home/user/list`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| role | String | 否 | 身份筛选 (student/teacher) |
| keyword | String | 否 | 关键字搜索（姓名、账号） |
| pageNum | Number | 否 | 页码，默认 1 |
| pageSize | Number | 否 | 每页条数，默认 10 |

**响应示例**:
```json
{
  "code": 200,
  "rows": [
    {
      "userId": 1,
      "userName": "zhangsan",
      "nickName": "张三",
      "roleType": "student",
      "role": "student",
      "className": "计算机1班",
      "classLevel": "A",
      "email": "zhangsan@example.com",
      "phonenumber": "13800138001",
      "createTime": "2026-01-15 10:30:00"
    }
  ],
  "total": 8
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| userId | Number | 用户ID |
| userName | String | 账号 |
| nickName | String | 姓名 |
| roleType | String | 身份类型 (student/teacher) |
| role | String | 角色标识 |
| className | String | 班级名称（学生） |
| classLevel | String | 班级等级（学生） |
| email | String | 邮箱 |
| phonenumber | String | 手机号 |
| createTime | String | 创建时间 |

---

### 2. 获取人员详情

**接口地址**: `GET /api/admin-home/user/:userId`

**请求头**:
```
Authorization: Bearer <token>
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|:----:|------|
| userId | Number | 用户ID |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "userId": 1,
    "userName": "zhangsan",
    "nickName": "张三",
    "roleType": "student",
    "role": "student",
    "className": "计算机1班",
    "classLevel": "A",
    "email": "zhangsan@example.com",
    "phonenumber": "13800138001",
    "createTime": "2026-01-15 10:30:00"
  }
}
```

---

### 3. 删除人员

**接口地址**: `DELETE /api/admin-home/user/:userId`

**请求头**:
```
Authorization: Bearer <token>
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|:----:|------|
| userId | Number | 用户ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "删除成功"
}
```

---

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
