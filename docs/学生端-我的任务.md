# 学生端 - 我的任务

## 功能概述

学生端任务模块提供任务管理功能，包括任务列表查看、做题、提交、查看结果等操作。

## 业务规则

- **未完成任务**：只显示**当前班级**的未完成任务（可做）
- **已完成任务**：可以查看**当前班级 + 历史班级**的所有已完成任务
- **做题权限**：只能做当前班级的任务，历史班级任务只能查看结果
- **过期处理**：任务过期后不能再做

---

## 页面功能

### 1. 任务列表
- 切换「未完成」/「已完成」标签查看不同状态的任务
- 未完成任务：显示当前班级未完成的任务
- 已完成任务：显示当前班级和历史班级的已完成任务
- 显示任务名称、类型、开始/截止时间、状态等信息

### 2. 去做任务
- 点击「去做」按钮进入做题页面
- 显示任务信息和题目列表
- 支持单选题、多选题、填空题
- 提交后自动批改并显示成绩

### 3. 查看详情
- 点击「查看详情」查看已完成任务的详细信息
- 显示得分、正确/错误题数、是否通过
- 显示每道题的正确答案和用户答案
- 显示题目解析

### 3. 提交任务答案

**接口地址**：`POST /api/student-tasks/:taskId/submit`

**功能说明**：提交任务答案，自动批改并记录成绩。只能提交当前班级的任务。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| taskId | number | 是 | 任务ID（路径参数） |
| answers | object | 是 | 答案对象，key为questionId，value为用户答案 |

**请求示例**：
```json
{
  "answers": {
    "1001": "A",
    "1002": ["A", "C"],
    "1003": "apple"
  }
}
```

**响应示例（通过）**：
```json
{
  "code": 200,
  "data": {
    "correctCount": 18,
    "wrongCount": 2,
    "totalQuestions": 20,
    "score": 90,
    "isPassed": true
  },
  "msg": "恭喜通过！"
}
```

**响应示例（未通过）**：
```json
{
  "code": 200,
  "data": {
    "correctCount": 10,
    "wrongCount": 10,
    "totalQuestions": 20,
    "score": 50,
    "isPassed": false
  },
  "msg": "未通过，请继续努力！"
}
```

**响应示例（非当前班级）**：
```json
{
  "code": 403,
  "msg": "只能提交当前班级的任务"
}
```

**响应示例（任务过期）**：
```json
{
  "code": 400,
  "msg": "任务已过期，无法提交"
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| correctCount | number | 正确题数 |
| wrongCount | number | 错误题数 |
| totalQuestions | number | 总题数 |
| score | number | 得分 |
| isPassed | boolean | 是否通过（60%及格） |

### 去做任务

```javascript
const startTask = async (record) => {
  const res = await fetch(`/api/student-tasks/${record.taskId}`, {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  const data = await res.json();
  
  if (data.code === 200) {
    doTaskModalVisible.value = true;
    currentTask.value = data.data;
  } else {
    message.error(data.msg);
  }
};
```

### 提交答案

```javascript
const submitTask = async () => {
  // 收集答案
  const answers = {};
  currentTask.value.questions.forEach(q => {
    answers[q.questionId] = q.userAnswer;
  });
  
  const res = await fetch(`/api/student-tasks/${currentTask.value.taskId}/submit`, {
    method: 'POST',
    headers: { 
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ answers })
  });
  
  const data = await res.json();
  if (data.code === 200) {
    message.success(data.msg);
    closeTaskModal();
    fetchTaskList(); // 刷新列表
  }
};
```

### 查看详情

const viewDetail = async (record) => {

### 详情页题目渲染

```vue
<template>
  <div v-for="(question, index) in currentTaskDetail" :key="question.questionId" class="detail-question-item">
    <div class="question-title">
      <span class="question-number">{{ index + 1 }}.</span>
      <span>{{ question.questionContent }}</span>
      <a-tag :color="question.isCorrect ? 'success' : 'error'" style="margin-left: 10px">
        {{ question.isCorrect ? '正确' : '错误' }}
      </a-tag>
    </div>
    
    <!-- 选择题选项 -->
    <div v-if="question.type === 'single' || question.type === 'multiple'" class="question-options">
      <div v-for="(option, optIndex) in question.options" :key="optIndex" class="option-item" 
           :class="{ 'correct-answer': isCorrectOption(question, option.key), 'user-answer': isUserOption(question, option.key) }">
        {{ option.key }}. {{ option.value }}
        <a-tag v-if="isCorrectOption(question, option.key)" color="success" style="margin-left: 10px">正确答案</a-tag>
        <a-tag v-if="isUserOption(question, option.key) && !isCorrectOption(question, option.key)" color="error" style="margin-left: 10px">您的答案</a-tag>
      </div>
    </div>
    
    <!-- 填空题答案 -->
    <div v-if="question.type === 'input'" class="answer-section">
      <div><strong>正确答案：</strong>{{ question.correctAnswer }}</div>
      <div><strong>您的答案：</strong><span :class="question.isCorrect ? 'correct-text' : 'wrong-text'">{{ question.userAnswer }}</span></div>
    </div>
    
    <!-- 解析 -->
    <div v-if="question.explanation" class="explanation-section">
      <strong>题目解析：</strong>
      <p>{{ question.explanation }}</p>
    </div>
  </div>
</template>

<script setup>
const isCorrectOption = (question, optionKey) => {
  const correct = question.correctAnswer;
  if (Array.isArray(correct)) {
    return correct.includes(optionKey);
  }
  return correct === optionKey;
};

const isUserOption = (question, optionKey) => {
  const userAnswer = question.userAnswer;
  if (Array.isArray(userAnswer)) {
    return userAnswer.includes(optionKey);
  }
  return userAnswer === optionKey;
};
</script>
```

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 302 | 重定向（历史班级任务引导到详情接口） |
| 400 | 请求参数错误或任务已过期/已完成 |
| 403 | 无权限（只能做当前班级任务） |
| 404 | 任务不存在 |
| 500 | 服务器内部错误 |
