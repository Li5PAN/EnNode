# 学生端 - 首页接口文档

## 概述

学生端首页相关接口，提供学习概览、打卡、任务完成情况等数据。

**基础路径**: `/api/student-home`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/overview` | GET | 获取学习概览数据 |
| 2 | `/clock-in-status` | GET | 获取打卡状态 |
| 3 | `/clock-in` | POST | 打卡 |
| 4 | `/weekly-tasks` | GET | 获取近7天任务完成情况 |
| 5 | `/weekly-words` | GET | 获取近7天单词学习趋势 |
| 6 | `/pending-tasks` | GET | 获取未完成任务列表 |
| 7 | `/class-progress` | GET | 获取班级任务完成进度 |
| 8 | `/class-notifications` | GET | 获取班级申请审核通知 |

---

## 接口详情

### 1. 获取学习概览数据

**接口地址**: `GET /api/student-home/overview`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "todayWordsLearned": 15,
    "totalWrongQuestions": 15,
    "classRank": 5,
    "pendingTasks": 3,
    "todayStudyMinutes": 45,
    "todayTasksCompleted": 2,
    "totalStudyMinutes": 1850,
    "masteredWords": 128,
    "tasksCompleted": 8,
    "totalWordsSearched": 256
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| todayWordsLearned | Number | 今日学习单词数 |
| totalWrongQuestions | Number | 累计错题数 |
| classRank | Number | 班级排名 |
| pendingTasks | Number | 待完成任务数 |
| todayStudyMinutes | Number | 今日学习时长(分钟) |
| todayTasksCompleted | Number | 今日完成任务数 |
| totalStudyMinutes | Number | 累计学习时长(分钟) |
| masteredWords | Number | 已掌握单词数 |
| tasksCompleted | Number | 累计完成任务数 |
| totalWordsSearched | Number | 累计查词次数 |

---

### 2. 获取打卡状态

**接口地址**: `GET /api/student-home/clock-in-status`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "clockedIn": true,
    "clockInDate": "2026-04-20",
    "consecutiveDays": 5,
    "totalDays": 16,
    "clockInDates": ["2026-03-03", "2026-03-08", ...]
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| clockedIn | Boolean | 今日是否已打卡 |
| clockInDate | String | 最后打卡日期 |
| consecutiveDays | Number | 连续打卡天数 |
| totalDays | Number | 累计打卡天数 |
| clockInDates | Array | 打卡日期列表 |

---

### 3. 打卡

**接口地址**: `POST /api/student-home/clock-in`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**: 无

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "clockedIn": true,
    "clockInDate": "2026-04-21",
    "consecutiveDays": 6,
    "totalDays": 17
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| clockedIn | Boolean | 打卡是否成功 |
| clockInDate | String | 打卡日期 |
| consecutiveDays | Number | 连续打卡天数 |
| totalDays | Number | 累计打卡天数 |

---

### 6. 获取未完成任务列表

**接口地址**: `GET /api/student-home/pending-tasks`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": [
    { "id": 1, "title": "第三单元单词测试", "deadline": "2026-04-25", "questionCount": 50, "urgency": "normal" },
    { "id": 2, "title": "阅读理解练习", "deadline": "2026-04-28", "questionCount": 30, "urgency": "normal" },
    { "id": 3, "title": "单词听写", "deadline": "2026-04-30", "questionCount": 40, "urgency": "comfortable" }
  ]
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| id | Number | 任务ID |
| title | String | 任务标题 |
| deadline | String | 截止日期 (YYYY-MM-DD) |
| questionCount | Number | 题目数量 |
| urgency | String | 紧急程度 (`urgent` / `normal` / `comfortable`) |

---

### 8. 获取班级申请审核通知

**接口地址**: `GET /api/student-home/class-notifications`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "type": "1",
      "typeText": "入班",
      "status": "approved",
      "content": "您申请加入【D级一班】班级的入班申请已通过",
      "className": "D级一班",
      "classLevel": "D",
      "targetClassName": null,
      "reason": null,
      "myReason": "我想加入这个班级学习",
      "createTime": "2026-04-20T10:30:00.000Z",
      "updateTime": "2026-04-20T14:00:00.000Z"
    },
    {
      "id": 2,
      "type": "3",
      "typeText": "换班",
      "status": "rejected",
      "content": "您申请从【D级一班】换到【C级一班】的换班申请被拒绝",
      "className": "D级一班",
      "classLevel": "D",
      "targetClassName": "C级一班",
      "reason": "任务完成率未达到100%",
      "myReason": "我想换到更高水平的班级",
      "createTime": "2026-04-19T09:00:00.000Z",
      "updateTime": "2026-04-19T16:30:00.000Z"
    }
  ]
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| id | Number | 申请ID |
| type | String | 申请类型：`1`=入班，`2`=退班，`3`=换班 |
| typeText | String | 申请类型文本 |
| status | String | 审核状态：`pending`=处理中，`approved`=已通过，`rejected`=未通过 |
| content | String | 通知内容描述 |
| className | String | 班级名称 |
| classLevel | String | 班级等级 |
| targetClassName | String | 目标班级名称（仅换班申请有值） |
| reason | String | 老师审核意见/拒绝原因 |
| myReason | String | 学生的申请理由 |
| createTime | String | 申请时间 |
| updateTime | String | 审核时间 |

---

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

---

## 模拟用户数据

接口会根据请求头中 token 解析出的用户名返回对应用户的数据：

| 用户名 | 说明 |
|--------|------|
| student | 默认学生用户 |
| lisi | 李四学生 |
| zhangsan | 张三学生 |

> 登录接口: `POST /api/auth/login`，请求体 `{ "username": "student", "password": "student" }`
