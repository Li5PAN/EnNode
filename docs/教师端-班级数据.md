 # 教师端 - 班级数据接口文档

## 概述

教师端班级数据统计相关接口，提供班级选择、任务完成对比、学生学习活跃分析、错题类型分析等数据图表功能。

**基础路径**: `/api/teacher-class-data`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/class-list` | GET | 获取班级下拉列表 |
| 2 | `/task-completion-chart` | GET | 获取班级任务完成对比图数据 |
| 3 | `/student-activity-chart` | GET | 获取学生学习活跃分析图数据 |
| 4 | `/error-type-chart` | GET | 获取学生错题类型分析图数据 |

---

## 接口详情

### 1. 获取班级下拉列表

**接口地址**: `GET /api/teacher-class-data/class-list`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "name": "A级-英语精英班",
      "level": "A"
    },
    {
      "id": 2,
      "name": "B级-英语进阶班",
      "level": "B"
    },
    {
      "id": 3,
      "name": "C级-英语提高班",
      "level": "C"
    },
    {
      "id": 4,
      "name": "D级-英语基础班",
      "level": "D"
    }
  ]
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| id | Number | 班级ID |
| name | String | 班级名称 |
| level | String | 班级等级 (A/B/C/D) |

### 2. 获取班级任务完成对比图数据

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|

**响应示例**:

```json
{

  "data": {
    "taskList": [
      {
        "taskId": 1,
        "taskName": "第三单元单词测试",
        "totalStudents": 25,

        "completionRate": 80

        "taskId": 2,
        "taskName": "第四单元单词测试",
        "totalStudents": 25,
        "completedCount": 22,
        "completionRate": 88
      },
      {
        "taskId": 3,
        "taskName": "基础词汇练习",
        "totalStudents": 25,
        "completedCount": 18,
        "completionRate": 72
      },
      {
        "taskId": 4,
        "taskName": "入门测试",
        "totalStudents": 25,
        "completedCount": 25,
        "completionRate": 100
      },
      {
        "taskId": 5,
        "taskName": "综合练习",

      "totalTasks": 5,
      "avgCompletionRate": 80,

      "lowestRate": 60
    }
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| taskList | Array | 任务列表 |
| taskList.taskId | Number | 任务ID |
| taskList.taskName | String | 任务名称 |
| taskList.totalStudents | Number | 班级总学生数 |
| taskList.completedCount | Number | 已完成学生数 |
| taskList.completionRate | Number | 完成率(%) |
| summary | Object | 汇总数据 |
| summary.totalTasks | Number | 任务总数 |
| summary.avgCompletionRate | Number | 平均完成率(%) |
| summary.highestRate | Number | 最高完成率(%) |
| summary.lowestRate | Number | 最低完成率(%) |

### 3. 获取学生学习活跃分析图数据

**接口地址**: `GET /api/teacher-class-data/student-activity-chart`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| classId | Number | 是 | 班级ID |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "days": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"],
    "completedTasks": [45, 52, 38, 61, 55, 72, 68],
    "exercises": [32, 41, 28, 45, 38, 58, 52],
    "studyDuration": [120, 145, 98, 168, 142, 195, 180],
    "summary": {
      "totalCompletedTasks": 391,
      "totalExercises": 294,
      "totalStudyDuration": 1048,
      "avgDailyActive": 35
    }
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| days | Array | 日期标签（最近7天） |
| completedTasks | Array | 每日完成任务数 |
| exercises | Array | 每日练习次数 |
| studyDuration | Array | 每日学习时长(分钟) |
| summary | Object | 汇总数据 |
| summary.totalCompletedTasks | Number | 总完成任务数 |
| summary.totalExercises | Number | 总练习次数 |
| summary.totalStudyDuration | Number | 总学习时长(分钟) |
| summary.avgDailyActive | Number | 日均活跃学生数 |

---

### 4. 获取学生错题类型分析图数据

**接口地址**: `GET /api/teacher-class-data/error-type-chart`

**请求头**:
```
Authorization: Bearer <token>
```

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| classId | Number | 是 | 班级ID |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "errorTypes": [
      { "type": "选择题", "count": 125, "percentage": 35.7 },
      { "type": "填空题", "count": 98, "percentage": 28.0 },
      { "type": "单词拼写", "count": 78, "percentage": 22.3 },
      { "type": "阅读理解", "count": 32, "percentage": 9.1 },
      { "type": "听力", "count": 17, "percentage": 4.9 }
    ],
    "summary": {
      "totalErrors": 350,
      "mostErrorType": "选择题",
      "mostErrorCount": 125
    },
    "trend": {
      "weeks": ["第1周", "第2周", "第3周", "第4周"],
      "errorCounts": [95, 88, 92, 75]
    }
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| errorTypes | Array | 错题类型分布 |
| errorTypes.type | String | 错题类型 |
| errorTypes.count | Number | 该类型错题数量 |
| errorTypes.percentage | Number | 占比(%) |
| summary | Object | 汇总数据 |
| summary.totalErrors | Number | 错题总数 |
| summary.mostErrorType | String | 最多错题类型 |
| summary.mostErrorCount | Number | 最多错题数量 |
| trend | Object | 趋势数据 |
| trend.weeks | Array | 周标签（最近4周） |
| trend.errorCounts | Array | 每周错题数量 |

---

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 模拟用户数据

接口会根据请求头中 token 解析出的用户名返回对应教师的数据：

| 用户名 | 说明 |
|--------|------|
| teacher | 张老师 |
| teacherLi | 李老师 |
| teacherWang | 王老师 |

> 登录接口: `POST /api/auth/login`，请求体 `{ "username": "teacher", "password": "teacher" }`
