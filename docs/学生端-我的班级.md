# 学生端 - 我的班级

## 功能概述

学生端班级模块提供班级选择、入班申请、班级信息查看、排行榜查看等功能。支持首次入班和重新入班两种场景，具有等级限制机制。

## 页面状态

### 1. 未入班状态
- 显示A/B/C/D级班级列表
- 首次入班提示：只能选择D级班级
- 重新入班提示：只能选择上次班级等级或更低等级
- 有待审核申请时显示提示信息

### 2. 已入班状态
- 显示班级详情信息
- 班级学习趋势图（近8周）
- 班级排行榜 Top 15
- 退出班级功能
- 换班功能（需任务完成率100%）

## 接口列表

### 1. 获取班级状态

**接口地址**：`GET /api/student-class/status`

**功能说明**：获取当前学生的班级状态，包括是否已入班、是否有待审核申请等。

**请求参数**：无

**响应示例**：
```json
{
  "code": 200,
  "data": {
    "status": 0,              // 状态：0-未入班，1-已入班，2-申请入班中，3-申请退班中，4-申请换班中
    "isFirstJoin": true,      // 是否首次入班
    "currentClassId": null,   // 当前班级ID（已入班时有值）
    "currentClassName": null, // 当前班级名称
    "pendingApplication": null, // 待审核申请信息
    "lastClassLevel": null    // 上次班级等级（退出班级后记录，用于限制重新入班）
  }
}
```

**状态说明**：
| 状态值 | 说明 |
|--------|------|
| 0 | 未入班 |
| 1 | 已入班 |
| 2 | 申请入班中 |
| 3 | 申请退班中 |
| 4 | 申请换班中 |

### 2. 获取班级列表

**接口地址**：`GET /api/student-class/list`

**功能说明**：获取所有可选班级列表，支持按等级筛选。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| level | string | 否 | 班级等级：A/B/C/D |

**响应示例**：
```json
{
  "code": 200,
  "rows": [
    {
      "id": 1,
      "classId": 1,
      "level": "A",
      "name": "A级-英语精英班",
      "teacher": "张教授",
      "teacherName": "张教授",
      "studentCount": 25,
      "memberCount": 25,
      "totalTasks": 20,
      "taskCount": 20
    }
  ],
  "total": 8
}
```

**字段说明**：
| 字段名 | 类型 | 说明 |
|--------|------|------|
| classId | number | 班级ID |
| level | string | 班级等级：A/B/C/D |
| name | string | 班级名称 |
| teacher | string | 授课老师 |
| teacherName | string | 授课老师名称 |
| studentCount | number | 班级人数 |
| memberCount | number | 班级成员数 |
| totalTasks | number | 任务总数 |
| taskCount | number | 任务数量 |

| 字段名 | 类型 | 说明 |

### 6. 申请入班

**接口地址**：`POST /api/student-class/apply`

**功能说明**：学生申请加入指定班级。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| classId | number | 是 | 要加入的班级ID |

**请求示例**：
```json
{
  "classId": 7
}
```

**响应示例（成功）**：
```json
{
  "code": 200,
  "msg": "成功加入班级",
  "data": {
    "classId": 7,
    "className": "D级-英语基础班"
  }
}
```

**响应示例（失败）**：
```json
{
  "code": 400,
  "msg": "首次入班只能选择D级班级"
}
```

**入班限制规则**：
1. 首次入班：只能选择D级班级
2. 重新入班：只能选择上次班级等级或更低等级
3. 已在班级中：需先退出当前班级
4. 有待审核申请：需等待审核完成

### 7. 退出班级

**接口地址**：`POST /api/student-class/quit`

**功能说明**：学生退出当前班级。

**请求参数**：无

**响应示例（成功）**：
```json
{
  "code": 200,
  "msg": "已成功退出班级"
}
```

**响应示例（失败）**：
```json
{
  "code": 400,
  "msg": "您当前不在任何班级中"
}
```

**注意事项**：
- 退出班级后，学习进度和排名数据将被清空
- 系统会记录上次班级等级，用于限制重新入班

**接口地址**：`POST /api/student-class/change`

**功能说明**：学生申请换到其他班级。

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| classId | number | 是 | 目标班级ID |

**请求示例**：
```json

  "classId": 5
}
```

**响应示例（成功）**：
```json
{
  "code": 200,
  "msg": "换班成功",
  "data": {
    "classId": 5,
    "className": "C级-英语提高班"
  }
}
```

**响应示例（失败）**：
```json

**换班限制规则**：
1. 必须已在班级中
2. 任务完成率必须达到100%
3. 只能换到同级或更低等级的班级

## 业务规则

### 班级等级说明

| 等级 | 说明 | 适用人群 |
|------|------|----------|
| A级 | 英语精英班 | 英语基础较好的学生 |
| B级 | 英语进阶班 | 有一定英语基础的学生 |
| C级 | 英语提高班 | 英语基础一般的学生 |
| D级 | 英语基础班 | 英语基础薄弱或新入学的学生 |

### 入班规则

1. **首次入班**
   - 新注册学生默认为首次入班状态
   - 首次入班只能选择D级班级
   - 入班后`isFirstJoin`变为`false`

2. **重新入班**
   - 退出班级后，系统记录上次班级等级
   - 重新入班只能选择同级或更低等级
   - 例如：上次是B级，只能选择B/C/D级

3. **换班**
   - 必须完成100%的班级任务
   - 只能换到同级或更低等级
   - 换班后排名和进度重新计算

### 等级顺序（从高到低）

```
A > B > C > D
```

  <a-button :type="selectedLevel === 'D' ? 'primary' : 'default'" @click="handleLevelChange('D')">D级</a-button>
</a-space>
```

### 班级卡片

显示信息：
- 班级等级标签（颜色区分）
- 班级名称
- 授课老师
- 班级人数
- 任务数量
- 申请按钮（根据规则禁用）

### 等级颜色映射

```javascript
const getLevelColor = (level) => {
  const colorMap = {
    'A': 'red',
    'B': 'orange',
    'C': 'blue',
    'D': 'green'
  };
  return colorMap[level] || 'default';
};
```

### 申请按钮状态

```javascript
const canApplyClass = (level) => {
  // 首次入班只能选D级
  if (isFirstJoin.value) {
    return level === 'D';
  }
  // 重新入班，只能选同级或更低
  if (lastClassLevel.value) {
    const levelOrder = { 'A': 1, 'B': 2, 'C': 3, 'D': 4 };
    return levelOrder[level] >= levelOrder[lastClassLevel.value];
  }
  return true;
};

const getApplyButtonText = (level) => {
  if (canApplyClass(level)) {
    return '申请入班';
  }
  return '不可选择';
};
```
