# 教师端—错题管理接口文档

## 概述

教师端错题管理相关接口，提供错题列表查询、详情查看、导入导出、删除等功能。

**基础路径**：`/api/teacher-home`

**认证方式**：除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/error/list` | GET | 获取错题列表（支持搜索筛选） |
| 2 | `/error/:questionId` | GET | 获取错题详情 |
| 3 | `/error/import` | POST | 导入错题（Excel文件） |
| 4 | `/error/export` | GET | 导出错题（Excel/PDF） |
| 5 | `/error/:questionId` | DELETE | 删除单条错题 |
| 6 | `/error/batch-delete` | POST | 批量删除错题 |
| 7 | `/error/template` | GET | 下载错题导入模板 |

---

### 1. 获取错题列表（支持搜索筛选）

Authorization: Bearer <token>

| classLevel | String | 否 | 班级等级（A/B/C/D） |
| pageNum | Number | 否 | 页码，默认 1 |

**响应示例**：

**字段说明**：

| 字段 | 类型 | 说明 |
|------|:----:|------|

| questionType | String | 题目类型（1—选择题 2—填空题 3—单词拼写） |
| questionContent | String | 题目内容 |

| mastered | Boolean | 是否已掌握 |
| errorDate | String | 错误日期 |

### 3. 导入错题（Excel文件）

**接口地址**：`POST /api/teacher-home/error/import`

**请求头**：
```
Authorization: Bearer <token>
Content-Type: multipart/form-data
```

**请求参数（FormData）**：

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| file | File | 是 | Excel文件（.xlsx, .xls） |

**Excel文件格式要求**：

| 列号 | 字段 | 说明 | 示例 |
|:----:|------|------|------|
| 1 | questionType | 题目类型（1—选择题 2—单词拼写 3—填空题） | 3 |
| 2 | taskName | 任务名称 | 定语从句专项练习 |
| 3 | errorDate | 错误日期 | 2026-03-11 17:28 |
| 4 | questionContent | 题目内容 | I still remember the days _____ we spent together. |
| 5 | correctAnswer | 正确答案 | which |
| 6 | wrongAnswer | 学生答案 | ex sed |
| 7 | wrongCount | 错误次数 | 1 |
| 8 | mastered | 是否已掌握（0—否 1—是） | 0 |

**响应示例**：
```json
{
  "code": 200,
  "msg": "成功导入 10 条错题记录"
}
```

**错误响应**：
```json
{
  "code": 400,
  "msg": "导入失败，文件格式不正确"
}
```

### 4. 导出错题（Excel/PDF）

**接口地址**：`GET /api/teacher-home/error/export`

**请求头**：
```
Authorization: Bearer <token>
```

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| format | String | 是 | 导出格式（excel/pdf） |
| scope | String | 否 | 导出范围（all—全部/filtered—筛选结果），默认 all |
| questionIds | String | 否 | 导出的错题ID列表，逗号分隔（仅当 scope=filtered 时使用） |
| startDate | String | 否 | 开始日期（筛选用） |
| endDate | String | 否 | 结束日期（筛选用） |
| questionType | String | 否 | 题目类型（筛选用） |
| classLevel | String | 否 | 班级等级（筛选用） |

**响应示例**：

- **Excel导出**：返回 .xlsx 文件下载
- **PDF导出**：返回 .pdf 文件下载

**导出字段**：

| 字段 | 说明 |
|------|------|
| 题目类型 | 选择题/填空题/单词拼写 |
| 任务名称 | 任务名称 |
| 错误日期 | 错题记录时间 |
| 题目内容 | 题目内容 |
| 正确答案 | 正确答案 |
| 学生答案 | 学生错误答案 |
| 错误次数 | 错误次数 |
| 班级等级 | A/B/C/D |
| 是否已掌握 | 是/否 |

### 5. 删除单条错题

**接口地址**：`DELETE /api/teacher-home/error/:questionId`

**请求头**：
```
Authorization: Bearer <token>
```

**路径参数**：

| 参数 | 类型 | 说明 |
|------|:----:|------|
| questionId | Number | 错题ID |

**响应示例**：
```json
{
  "code": 200,
  "msg": "删除成功"
}
```

**错误响应**：
```json
{
  "code": 404,
  "msg": "错题记录不存在"
}
```

---

### 6. 批量删除错题

**接口地址**：`POST /api/teacher-home/error/batch-delete`

**请求头**：
```
Authorization: Bearer <token>
Content-Type: application/json
```

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| questionIds | Array | 是 | 错题ID数组 |

**请求示例**：
```json
{
  "questionIds": [1, 2, 3, 4, 5]
}
```

**响应示例**：
```json
{
  "code": 200,
  "msg": "成功删除 5 条错题记录"
}
```

### 7. 下载错题导入模板

**接口地址**：`GET /api/teacher-home/error/template`

**请求头**：
```

```

**响应示例**：

返回 .xlsx 模板文件下载

**模板内容示例**：

| 1 | 词汇测试 | 2026-03-10 15:20 | What is the antonym of 'beautiful'? | C | A | 1 | 0 |

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 400 | 请求参数错误或文件格式不正确 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
