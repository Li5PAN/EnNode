# 学生端 - 单词学习 API 文档

## 概述

单词学习模块提供单词浏览、搜索、收藏、学习练习等功能。所有接口需要携带 Token 认证。

**基础路径**: `/api/student-words`

**认证方式**: Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

### 1. 获取单词列表（分页+搜索）

```
GET /api/student-words
```

**Query 参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 否 | 搜索关键词（支持中英文） |
| page | number | 否 | 页码，默认 1 |
| pageSize | number | 否 | 每页数量，默认 10 |

**请求示例**:
```
GET /api/student-words?keyword=学习&page=1&pageSize=10
```

**响应示例**:
```json
{
  "code": 200,
  "total": 20,
  "page": 1,
  "pageSize": 10,
  "rows": [
    {
      "word": "study",
      "wordId": 4,
      "ukPhonetic": "ˈstʌdi",
      "usPhonetic": "ˈstʌdi",
      "meanings": ["n. 学习，研究", "vt. 学习；考虑"],
      "englishWord": "study",
      "chineseMeaning": "n. 学习，研究；vt. 学习；考虑"
    }
  ],
  "records": []
}
```

### 2. 搜索单词（支持中英文）

```
GET /api/student-words/search
```

**Query 参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 是 | 搜索关键词 |

**请求示例**:
```
GET /api/student-words/search?keyword=苹果
```

**响应示例**:
```json
{
  "success": true,
  "data": {
    "word": "apple",
    "wordId": 2,
    "ukPhonetic": "ˈæpl",
    "usPhonetic": "ˈæpl",
    "meanings": ["n. 苹果；苹果树；苹果公司"],
    "examples": [
      {
        "sentence": "An apple a day keeps the doctor away.",
        "translation": "一天一苹果，医生远离我。"
      }
    ]
  }
}
```

**未找到时**:
```json
{
  "success": false,
  "message": "未找到"苹果"相关的单词"
}
```

### 3. 收藏单词

**Body 参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| wordId | number | 是 | 单词ID |

**请求示例**:
```json
{

}
```

**响应示例**:
```json
{

  "msg": "收藏成功"
}
```

**已收藏时**:
```json

}
```

## 单词数据结构

```typescript
interface Word {
  word: string;              // 英文单词
  wordId: number;            // 单词ID
  ukPhonetic: string;        // 英式音标
  usPhonetic: string;        // 美式音标
  ukSpeech: string;          // 英式发音URL
  usSpeech: string;          // 美式发音URL
  meanings: string[];        // 释义列表
  webMeanings: {             // 网络释义
    phrase: string;
    meaning: string;
  }[];
  examples: {                // 例句
    sentence: string;
    translation: string;
    sentenceSpeech: string;
  }[];
  wordForms: string[];       // 词形变化
}
```

## 响应码说明

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或 Token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 搜索功能说明

- **中文搜索**: 支持精确中文（如"苹果"）和模糊匹配（搜索含义中包含的中文）
- **英文搜索**: 支持精确匹配和模糊匹配（单词���释义中包含关键词）
- **搜索示例**:
  - `keyword=苹果` → 返回 apple
  - `keyword=学习` → 返回 study, learn
  - `keyword=hello` → 返回 hello
  - `keyword=stu` → 返回 study, student
