# 管理员端 - 首页接口文档

## 概述

管理员端首页相关接口，提供数据概览、班级分布、用户增长趋势、换班/退班/班级创建趋势等数据。

**基础路径**: `/api/admin-home`

**认证方式**: 除特别说明外，所有接口需在 Header 中携带 `Authorization: Bearer <token>`

---

## 接口列表

| 序号 | 接口 | 方法 | 说明 |
|:---:|------|:----:|------|
| 1 | `/overview` | GET | 获取数据概览 |
| 2 | `/level-distribution` | GET | 获取各等级班级分布 |
| 3 | `/user-growth-trend` | GET | 获取用户增长趋势 |
| 4 | `/class-change-trend` | GET | 获取换班变化趋势 |
| 5 | `/drop-class-trend` | GET | 获取退班变化趋势 |
| 6 | `/class-create-trend` | GET | 获取班级创建趋势 |

---

## 接口详情

### 1. 获取数据概览

**接口地址**: `GET /api/admin-home/overview`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "totalUsers": 1250,
    "totalStudents": 980,
    "totalTeachers": 45,
    "totalClasses": 68,
    "activeToday": 356,
    "pendingApplications": 12,
    "monthNewStudents": 85,
    "monthNewTeachers": 5,
    "monthNewClasses": 8
  }
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| totalUsers | Number | 用户总数 |
| totalStudents | Number | 学生总数 |
| totalTeachers | Number | 教师总数 |
| totalClasses | Number | 班级总数 |
| activeToday | Number | 今日活跃用户数 |
| pendingApplications | Number | 待审批申请数 |
| monthNewStudents | Number | 本月新增学生数 |
| monthNewTeachers | Number | 本月新增教师数 |
| monthNewClasses | Number | 本月新增班级数 |

---

### 2. 获取各等级班级分布

**接口地址**: `GET /api/admin-home/level-distribution`

**请求头**:
```
Authorization: Bearer <token>
```

**响应示例**:
```json
{
  "code": 200,
  "data": [
    { "level": "A级", "classCount": 12, "studentCount": 320, "percentage": 18 },
    { "level": "B级", "classCount": 18, "studentCount": 480, "percentage": 27 },
    { "level": "C级", "classCount": 22, "studentCount": 550, "percentage": 31 },
    { "level": "D级", "classCount": 16, "studentCount": 380, "percentage": 24 }
  ]
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| level | String | 班级等级 (A/B/C/D) |
| classCount | Number | 该等级班级数量 |
| studentCount | Number | 该等级学生总数 |
| percentage | Number | 占比(%) |

---

**响应示例**:

```json
    { "date": "2026-04-15", "newStudents": 18, "newTeachers": 2, "newUsers": 20 },
    { "date": "2026-04-16", "newStudents": 15, "newTeachers": 0, "newUsers": 15 },
    { "date": "2026-04-17", "newStudents": 22, "newTeachers": 1, "newUsers": 23 },
    { "date": "2026-04-18", "newStudents": 20, "newTeachers": 2, "newUsers": 22 },
    { "date": "2026-04-19", "newStudents": 25, "newTeachers": 1, "newUsers": 26 },
    { "date": "2026-04-20", "newStudents": 30, "newTeachers": 2, "newUsers": 32 }
  ]
}
```

**字段说明**:

| 字段 | 类型 | 说明 |
|------|:----:|------|
| date | String | 日期 |
| newStudents | Number | 新增学生数 |
| newTeachers | Number | 新增教师数 |
| newUsers | Number | 新增用户总数 |

## 响应码说明

| code | 说明 |
|:----:|------|
| 200 | 请求成功 |
| 401 | 未登录或 token 无效 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
